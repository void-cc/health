# Phased Delivery Roadmap

## Phase 0 - Discovery and Audit

- Deliverables:
  - screen registry
  - CRUD maturity matrix
  - interaction contract ledger
  - clinical UX risk register
- Exit criteria:
  - full route/control coverage
  - prioritized backlog with acceptance criteria

## Phase 1 - Clinical Foundations

- Deliverables:
  - canonical model definitions
  - tenancy and validation strategy
  - baseline triage dashboard architecture
  - copy and range terminology normalization
- Exit criteria:
  - interpretable risk/trend states for core metrics

## Phase 2 - Interactive Insights and Decision Support

- Deliverables:
  - reusable chart interaction system
  - review queues and escalation flows
  - control remediation implementation for high-impact actions
- Exit criteria:
  - clinician can navigate from alert to action in <=3 steps

## Phase 3 - Hardening and Productization

- Deliverables:
  - lean quality gates and observability
  - async feedback consistency
  - CSS/JS duplication cleanup
- Exit criteria:
  - stable fast PR lane <=10 minutes
  - no high-impact dead controls
