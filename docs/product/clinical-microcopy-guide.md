# Clinical Microcopy Guide

## Canonical Terms

- Use **expected range** (not “normal range” or “range” in UI labels).
- Use **needs review** for attention state (not “out of range” unless clinically required context).
- Use sentence case for action labels (`Add blood test`, `Record vitals`, `Show plot`).

## Status Vocabulary

- `Normal`: value within expected range.
- `Needs review`: value outside expected range or trend concern.
- `No reference range`: interpretation unavailable.

## Abbreviations

- `SpO2`: peripheral oxygen saturation (tooltips on first-use contexts).
- `BBT`: basal body temperature (tooltips on first-use contexts).

## Interaction Messaging

- Loading: “Preparing …”
- Success: past-tense confirmation (“… exported.”, “… saved.”)
- Error: neutral and actionable (“… failed. Please try again.”)

## Do/Do Not

- Do prioritize calm, precise, non-alarmist tone.
- Do state next action when possible.
- Do not use mixed terminology for the same clinical concept.
