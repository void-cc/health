Supported Formats

Upload a CSV, JSON, PDF, or HL7 file containing your health data. The CSV file must contain a header row. JSON files should be a list of objects or a valid FHIR Bundle. HL7 files (.hl7) are parsed for OBR and OBX segments.

Tip: Use Export Data to get a correctly formatted file, modify it, and import it back.

  • Date — YYYY-MM-DD format
  • Type — 'Blood Test' or 'Vitals'
  • Name — Test name or 'Vital Signs'
  • Value — Numeric, or comma-separated string for vitals (e.g. "70.0 kg, 65 bpm, 120/80")
