Import Data

Upload a CSV, JSON, PDF, or HL7 file containing your health data.

Expected Format

The CSV file must contain a header row. JSON files should contain a list of objects with the corresponding keys, or be a valid FHIR Bundle. HL7 files (.hl7) are automatically parsed for OBR and OBX segments.

Tip: The easiest way to get the correct format is to use the Export Data feature, modify the resulting file, and import it back.

  • Date (YYYY-MM-DD)
  • Type ('Blood Test' or 'Vitals')
  • Name (Test name or 'Vital Signs')
  • Value (Numeric for tests, or a comma-separated string for vitals e.g. "70.0 kg, 65 bpm, 120/80")
