Upload a CSV file containing your health data.

Expected CSV Format

The CSV file must contain a header row. The required columns depend on the data type.

Tip: The easiest way to get the correct format is to use the Export Data feature, modify the resulting file, and import it back.

  • Date (YYYY-MM-DD)
  • Type ('Blood Test' or 'Vitals')
  • Name (Test name or 'Vital Signs')
  • Value (Numeric for tests, or a comma-separated string for vitals e.g. "70.0 kg, 65 bpm, 120/80")
